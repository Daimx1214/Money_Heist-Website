# 🎭 Money Heist Website

A premium cinematic fan-made website inspired by **Money Heist (La Casa de Papel)** featuring immersive animations, interactive sections, modern UI design, and smooth scrolling effects.

## 🌟 Features

- 🎬 Cinematic Hero Section
- ⚡ Smooth Scrolling Experience
- 🎨 Modern Responsive Design
- 🎭 Money Heist Character Showcase
- 📖 Story & Timeline Sections
- 🖼️ Interactive Image Gallery
- 🎥 Trailer & Video Sections
- 📊 Statistics Dashboard
- 💬 Quotes & Reviews Section
- 🌌 Three.js Background Effects
- ✨ Particle Animations
- 📱 Fully Responsive Layout
- 🚀 Fast and Lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP Animation Library
- ScrollTrigger
- Lenis Smooth Scrolling
- Three.js
